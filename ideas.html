This project analyzes how different learning strategies relate to perceived academic performance in university students. Through statistical analysis and a linear regression model, the most relevant predictors influencing this perception are identified.

This project uses deep learning to classify human facial expressions from black and white images. The goal is to build a model capable of identifying emotions such as happiness, sadness, anger, surprise, and others, and then apply it in real time using a webcam.
